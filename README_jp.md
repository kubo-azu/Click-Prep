# 🛟 Click-Prep ユーザーガイド 🛟
Click-PrepはRとShinyで構築されたインタラクティブなWebアプリケーションです。qPCR装置からエクスポートされた表形式ファイルを[Click-qPCR](https://kubo-azu.shinyapps.io/Click-qPCR/)での解析用に変換・クリーニングできるよう設計されています。

直感的なGUIを備えたClick-Prepは，Excelでのコピー＆ペーストや複数のプレートデータセットの手動マージといった煩雑なデータ前処理手順を不要にします。

Click-Prepは [https://kubo-azu.shinyapps.io/Click-Prep/](https://kubo-azu.shinyapps.io/Click-Prep/) から誰でも無料で利用でき，ローカルインストールは不要です。

## 🌟 主な機能
インタラクティブな列マッピング：シンプルなドロップダウンメニューを使用し，機器固有の列ヘッダーを必要なカテゴリ（sample, group, gene, Cq）に簡単にリンクできます。

手動グループ割り当て：生データにgroup列がない場合，ツール上で実験条件を定義してサンプルに直接割り当てることができます。

平均Cq値計算：テクニカルレプリケートが存在する場合の平均Cq値を自動的に計算します。

複数CSVファイルの統合：複数のClick-qPCR用フォーマット済みCSVファイルをマージし，単一の統合データセットCSVを作成します。複数qPCRプレートにわたる大規模解析などに利用可能です。

## 📖 ワークフロー
アプリケーションは、データフォーマットプロセスを順を追ってガイドする4つのタブで構成されています。

#### <b>Introduction:</b>

アプリの概要，基本的な操作手順，プライバシー/セキュリティポリシーを表示しています。

#### <b>File Trimming:</b>

生のqPCR出力ファイル（.csv, .txt, .tsv, .xlsx）をアップロードしてください。Skipオプションを使用すると，メタデータ行をスキップできます。列を必要なフォーマットにマッピングし，必要に応じてグループを手動で割り当ててください。マッピング要件がすべて満たされるとステータスパネルに通知が表示されます。

#### <b>Mean Cq Calculation & Download:</b>

列マッピングが完了したら、このタブに進み、計算結果をプレビューしてください。「フォーマット済みCSV（全反復）」または平均値の「平均Cq値CSV」のいずれかをダウンロードできます。

#### <b>Advanced - Input CSV Integration:</b>

複数のフォーマット済みCSVファイル（例：別々のqPCRプレートから得られたファイル）をアップロードすることで，Click-qPCR解析に適した単一のCSVファイルを作成できます。

## 🔒 プライバシーとセキュリティポリシー
Click-Prepは，臨床サンプルや企業独自のデータなど機密性の高い研究データを安全に取り扱うことを想定し，厳格なデータプライバシー保護を念頭に置いて設計されています。

#### <b>インメモリ処理:</b>

アップロードされたファイルは，セッション中はサーバーのアクティブメモリ内でのみ処理されます。サーバー上にファイルが永続的に保存されることはありません。


#### <b>自動データ消去:</b>

すべての一時ファイルとキャッシュデータは，処理完了後またはブラウザタブを閉じた直後に自動的に完全に削除されます。

#### <b>追跡・共有なし:</b>

入力データ，分析条件，フォーマット結果は厳重に機密保持されます。データの収集，ログ記録，第三者への共有は一切行えません。
